================================================================================

View Application Logs:

  aws ecs describe-services \
    --cluster kerala-toors-cluster \
    --services kerala-toors-service

Scale Service:
  aws ecs update-service \
    --cluster kerala-toors-cluster \
    --service kerala-toors-service \
    --desired-count 4

  terraform output -raw application_url
